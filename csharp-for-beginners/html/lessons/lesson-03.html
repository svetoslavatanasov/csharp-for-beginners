Променливи и типове данни

Какво ще научиш

  • Какво е променлива.
  • Как да създадеш променлива за текст, цяло число и дробно число.
  • Как да разпознаваш тип, име и стойност.
  • Как да показваш и променяш стойност на променлива.
  • Как да избираш подходящ тип за прости данни.

Защо това е важно

В Урок 2 програмите ни показваха готов текст. Сега започваме да пазим информация: име на играч, точки, температура, цена или съобщение.

Променливите са първата стъпка към програма, която използва данни, вместо само да показва предварително написани редове.

История с Byte

Byte подрежда малки кутии. На първата пише name, а вътре има "Мая". На втората пише score, а вътре има 15.

Byte: "Когато кутията има име, мога да намеря стойността по-късно. Но трябва да знам и какъв вид стойност има вътре."

Точно това правят променливите. Те са именувани места за пазене на стойности.

Основно обяснение

Променлива е именувано място за пазене на стойност.

string name = "Мая";
  • string е типът данни. Той казва, че стойността е текст.
  • name е името на променливата.
  • = означава "сложи тази стойност в тази променлива".
  • "Мая" е стойността.
  • ; завършва инструкцията.
сложи стойността отдясно в променливата отляво

Първите типове данни

Тип За какво служи Пример
string текст "Иван"
int цяло число 15
double дробно число 22.5 
string city = "София";
int age = 15;
double temperature = 22.5;

Визуално обяснение

+----------------+
| name           |
| "Мая"          |
+----------------+

+----------------+
| score          |
| 15             |
+----------------+

+----------------+
| temperature    |
| 22.5           |
+----------------+

Името е етикетът на кутията. Стойността е това, което е вътре. Типът казва какъв вид нещо очакваме вътре.

[ тип ] [ име ] = [ стойност ];

string  name    = "Мая";
int     score   = 15;
double  height  = 1.72;

Първи пример

string robotName = "Byte";
int batteryLevel = 87;
double temperature = 22.5;

Console.WriteLine("Име:");
Console.WriteLine(robotName);

Console.WriteLine("Батерия:");
Console.WriteLine(batteryLevel);

Console.WriteLine("Температура:");
Console.WriteLine(temperature);

Резултат:

Име:
Byte
Батерия:
87
Температура:
22.5

Console.WriteLine(robotName); показва стойността на променливата. Console.WriteLine("robotName"); показва самия текст robotName.

Разбивка на кода

int batteryLevel = 87;

Този ред създава променлива за цяло число, нарича я batteryLevel и слага вътре стойността 87.

Console.WriteLine(batteryLevel);

Този ред намира променливата, взема стойността вътре и я показва.

Промяна на стойност

int points = 10;
Console.WriteLine(points);

points = 15;
Console.WriteLine(points);

Резултат:

10
15

Първия път създаваме променливата с int. Втория път само сменяме стойността, затова не пишем типа отново.

Чести грешки

Текст без кавички

string name = Byte;

Поправка:

string name = "Byte";

Число като текст

int age = "15";

int очаква цяло число, а "15" е текст.

int age = 15;

Дробно число с грешен тип

int temperature = 22.5;

Поправка:

double temperature = 22.5;

Използване преди създаване

Console.WriteLine(score);
int score = 20;

Поправка:

int score = 20;
Console.WriteLine(score);

Показване на името вместо стойността

int score = 20;
Console.WriteLine("score");

За стойността махаме кавичките:

Console.WriteLine(score);

Как мисли програмистът

Преди да създадеш променлива, попитай: каква информация пазя, какъв тип е тя, какво ясно име ще използвам и дали стойността ще се промени.

Искам да пазя точки в игра.
Точките са цяло число.
Името score е кратко и ясно.
int score = 0;
Byte: "Етикетът е ясен. Типът е ясен. Стойността е ясна. Мога да работя."

Практика

Какво ще отпечата тази програма?

string playerName = "Ива";
int points = 12;

Console.WriteLine(playerName);
Console.WriteLine(points);

Предвиди стойността

Каква е стойността на coins накрая?

int coins = 5;
coins = 8;
coins = 10;

Намери бъга

int age = "15";
Console.WriteLine(age);

Поправи кода

string player name = "Иво";
int score = "20";

Console.WriteLine("playerName");
Console.WriteLine("score");

Кодът трябва да покаже Иво и 20.

Обобщение

Променливата е именувано място за пазене на стойност. string пази текст, int пази цяло число, а double пази дробно число.

Типът казва какво може да има в кутията,
името казва как да я намерим,
а стойността е това, което пазим вътре.
