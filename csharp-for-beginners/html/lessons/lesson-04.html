Вход от клавиатурата и преобразуване на типове

Какво ще научиш

  • Как да прочетеш текст от клавиатурата с Console.ReadLine.
  • Как да запазиш въведеното в променлива.
  • Защо входът първо идва като текст.
  • Как да преобразуваш текст към int и double.
  • Как да пишеш малки интерактивни конзолни програми.

Защо това е важно

Досега програмите използваха стойности, които бяха написани предварително в кода. С вход от клавиатурата програмата може да попита човека за име, възраст, точки или цена.

Така програмата вече не само говори. Тя слуша.

История с Byte

Byte иска да поздрави човека по име. Вместо винаги да показва "Здравей, човек!", той пита дали може първо да прочете име.

Byte: "Мога ли първо да попитам за име, после да използвам отговора?"

Да. За това ни трябва Console.ReadLine(). Но Byte трябва да помни, че всичко от клавиатурата първо пристига като текст.

Основно обяснение

За да прочетем ред от клавиатурата, използваме:

Console.ReadLine()

Най-често пазим резултата в променлива:

string name = Console.ReadLine();
Изчакай човекът да напише нещо.
Когато натисне Enter, вземи написания текст.
Сложи го в променливата name.
Console.WriteLine("Как се казваш?");
string name = Console.ReadLine();

Console.WriteLine("Здравей!");
Console.WriteLine(name);

Входът е текст

Console.ReadLine() връща текст. Това е правилно:

string ageText = Console.ReadLine();

Това не е правилно:

int age = Console.ReadLine();
Console.ReadLine() дава string.
int очаква цяло число.

Визуално обяснение

[ клавиатура ]
      |
      v
Console.ReadLine()
      |
      v
[ string променлива ]
човекът пише: 15
      |
      v
Console.ReadLine()
      |
      v
ageText = "15"

Ако ни трябва число, добавяме преобразуване:

ageText = "15" -> int.Parse(ageText) -> age = 15
"15"   е текст
15     е число

Първи пример

Console.WriteLine("Как се казваш?");
string name = Console.ReadLine();

Console.WriteLine("Здравей!");
Console.WriteLine(name);

Примерно изпълнение:

Как се казваш?
Иво
Здравей!
Иво

Разбивка на кода

Console.WriteLine("Как се казваш?");

Това показва въпрос.

string name = Console.ReadLine();

Това прочита отговора и го пази като текст.

Console.WriteLine(name);

Това показва стойността, която човекът е въвел.

Преобразуване към число

Console.WriteLine("На колко години си?");
string ageText = Console.ReadLine();
int age = int.Parse(ageText);

int nextYear = age + 1;

Console.WriteLine("След една година ще си на:");
Console.WriteLine(nextYear);

Ако човекът въведе 15, резултатът е 16. В следващия урок ще разгледаме операторите по-спокойно.

Преобразуване към дробно число

Console.WriteLine("Въведи височина в метри:");
string heightText = Console.ReadLine();
double height = double.Parse(heightText);

Console.WriteLine("Въведената височина е:");
Console.WriteLine(height);

В примерите от курса ще пишем дробни числа с точка, например 1.75. Ако в твоята среда това не се приеме, пробвай със запетая: 1,75. Засега не е нужно да помниш защо има разлика. Важното е идеята: текстът трябва да стане дробно число.

Чести грешки

Опит да сложим текст директно в int

int age = Console.ReadLine();

Поправка:

string ageText = Console.ReadLine();
int age = int.Parse(ageText);

Преобразуване на дума към число

string ageText = "петнадесет";
int age = int.Parse(ageText);

int.Parse очаква текст, който изглежда като цяло число, например "15".

Кавички около името на променливата

string ageText = "15";
int age = int.Parse("ageText");

Поправка:

int age = int.Parse(ageText);

Забравени скоби

string name = Console.ReadLine;

Поправка:

string name = Console.ReadLine();

Очакваме програмата да поправи входа сама

Ако програмата пита за число, а човекът напише много, int.Parse няма да знае как да направи число от това. По-късно, когато учим условия, ще проверяваме входа по-внимателно.

Как мисли програмистът

1. Какво трябва да попитам?
2. Какво ще въведе човекът?
3. Текст ли ми трябва, или число?
4. Ако ми трябва число, кога ще преобразувам текста?
5. Как ще използвам стойността?
Byte: "Първо слушам. После решавам дали текстът трябва да стане число."

Практика

Какво ще отпечата тази програма?

Потребителят въвежда Ива.

Console.WriteLine("Име?");
string name = Console.ReadLine();
Console.WriteLine("Здравей!");
Console.WriteLine(name);

Предвиди стойността

Потребителят въвежда 14. Каква е стойността на nextYear?

string ageText = Console.ReadLine();
int age = int.Parse(ageText);
int nextYear = age + 1;

Намери бъга

Console.WriteLine("На колко години си?");
int age = Console.ReadLine();

Console.WriteLine(age);

Поправи кода

Console.WriteLine("Колко точки имаш?");
string pointsText = Console.ReadLine();
int points = int.Parse("pointsText");

Console.WriteLine("Точки:");
Console.WriteLine("points");

Обобщение

Console.ReadLine() чете един ред от клавиатурата. Входът първо идва като string. За числа използваме int.Parse или double.Parse.

Входът от клавиатурата започва като текст.
Преобразуваме го само когато ни трябва число.
