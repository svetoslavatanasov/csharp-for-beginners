Първа C# програма

Какво ще научиш

  • Какво е конзолна програма.

  • Какво прави Console.WriteLine.
  • Как да откриваш малки синтактични грешки.

Защо това е важно

Ако можеш да покажеш текст, можеш да видиш какво прави програмата. Това е първата стъпка към тестване, мислене и поправяне на код.

История с Byte

Byte получава първата истинска C# инструкция:

Console.WriteLine("Здравей, Byte!");
Byte: "Виждам точна инструкция. Има действие, скоби, текст в кавички и знак за край."

Основно обяснение

Конзолата е текстов прозорец, в който програмата може да показва текст.

+----------------------------------+
| Конзола                          |
|                                  |
| Здравей, свят!                   |
|                                  |
+----------------------------------+

Първата C# инструкция е:

Console.WriteLine("Здравей, свят!");
  • Console е свързано с конзолата.
  • WriteLine означава "напиши ред".
  • Текстът в кавички е това, което ще се покаже.
  • ; завършва инструкцията.

Визуално обяснение

[ C# инструкция ] -> [ компютърът я изпълнява ] -> [ текст в конзолата ]
Console.WriteLine("Здравей!");
          |
          v
+----------------+
| Здравей!       |
+----------------+

Първи пример

Console.WriteLine("Здравей!");
Console.WriteLine("Аз уча C#.");
Console.WriteLine("Byte също е тук.");

Резултат:

Здравей!
Аз уча C#.
Byte също е тук.

Разбивка на кода

Console.WriteLine("Здравей, Byte!");
  • Console казва, че работим с конзолата.
  • . свързва Console с WriteLine.
  • "Здравей, Byte!" е текстът за показване.
  • ; казва, че инструкцията свършва.

Чести грешки

Липсваща точка и запетая

Console.WriteLine("Здравей")

Поправка:

Console.WriteLine("Здравей");

Липсваща кавичка

Console.WriteLine("Здравей);

Поправка:

Console.WriteLine("Здравей");

Грешни главни и малки букви

console.writeline("Здравей");

C# прави разлика между главни и малки букви.

Как мисли програмистът

прочети -> предвиди -> стартирай -> сравни

Преди да стартираш код, опитай да предвидиш какво ще покаже.

Практика

Какво ще отпечата тази програма?

Console.WriteLine("Старт");
Console.WriteLine("Byte проверява инструкциите.");
Console.WriteLine("Готово");

Намери бъга

Console.WriteLine("Здравей, Byte!")
Console.WriteLine("Добре дошъл.");

Поправи кода

Console.WriteLine(Аз уча C#.);
Console.WriteLine("Това е първата ми програма.)

Упражнения по трудност

⭐ Easy

  1. Напиши инструкция, която показва Здравей!.
  2. Промени текста така, че програмата да показва твоето име.
  3. Предвиди изхода на три поредни Console.WriteLine инструкции.

⭐⭐ Medium

  1. Напиши програма с 4 реда, която показва кратка визитка.
  2. Подреди три инструкции така, че да покажат "Първо", "После", "Накрая".

⭐⭐⭐ Hard

  1. Напиши програма, която показва малка ASCII картичка.
  2. Предвиди изхода на програма с 4 реда, без да я стартираш.

Обобщение

Console.WriteLine показва един ред текст. C# има точен синтаксис, а програмата изпълнява инструкциите отгоре надолу.
