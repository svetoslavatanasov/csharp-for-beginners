# Визуален стил

Курсът използва ASCII диаграми, защото те работят добре в Markdown и HTML.

## Общи правила

- Използвай прости линии, стрелки и кутии.
- Показвай една идея на диаграма.
- Не прави диаграмите по-широки от около 80 символа.
- След всяка диаграма добавяй кратко обяснение.

## HTML версии

В HTML уроците всички ASCII диаграми, псевдокод и C# примери трябва да бъдат в:

```html
<pre><code>...</code></pre>
```

Всеки HTML урок трябва да използва:

```html
<link rel="stylesheet" href="../assets/course.css">
```

Не добавяй отделен CSS за всеки урок. Подобрявай общия файл `html/assets/course.css`.

## Променливи като кутии

```text
+---------+
| score   |
| 15      |
+---------+
```

## Поток на програма

```text
[ начало ]
    |
    v
[ стъпка ]
    |
    v
[ край ]
```

## Условия

```text
       +-------------+
       | age >= 13 ?|
       +-------------+
        | да    | не
        v       v
   [ позволи ] [ откажи ]
```

## Цикли

```text
[ провери условие ]
        |
        v
[ изпълни стъпка ]
        |
        +---- обратно към проверката
```
