# Правила за упражнения

Упражненията трябва да изграждат увереност и да тренират мислене.

## Нива

- ⭐ Easy - директно упражнение върху новата идея.
- ⭐⭐ Medium - комбинира новата идея с вече научени неща.
- ⭐⭐⭐ Hard - изисква повече стъпки или внимателно проследяване.
- 🧠 Challenge - незадължителна задача за любопитни ученици.

## Категории

Всеки урок трябва да има:

- "Какво ще отпечата?";
- "Предвиди стойността";
- "Намери бъга";
- "Поправи кода";
- упражнения по трудност;
- мини проект.

## Добри задачи

Добрата задача е конкретна, кратка и има ясен очакван резултат.

Вместо:

```text
Обработи данните.
```

Напиши:

```text
Попитай за 3 оценки и покажи средната стойност.
```

## Решения

Решенията трябва да показват мислене:

1. Идея с човешки думи.
2. Стъпки.
3. Код.
4. Обяснение.
5. Как да провериш резултата.
