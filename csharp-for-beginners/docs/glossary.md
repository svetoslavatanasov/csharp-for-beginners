# Речник

| Български | English | Кратко обяснение |
|---|---|---|
| програма | program | Набор от инструкции, които компютърът изпълнява. |
| алгоритъм | algorithm | Последователност от стъпки за решаване на задача. |
| променлива | variable | Именувано място за пазене на стойност. |
| стойност | value | Данни, например `15` или `"Иван"`. |
| тип данни | data type | Вид стойност, например число или текст. |
| низ | string | Текстова стойност. |
| цяло число | integer | Число без дробна част. |
| условие | condition | Въпрос в кода с отговор `true` или `false`. |
| цикъл | loop | Код, който се повтаря. |
| метод | method | Именуван блок код. |
| клас | class | Шаблон за създаване на обекти. |
| обект | object | Конкретно нещо, създадено по клас. |
| грешка | error | Проблем, който пречи на програмата. |
| бъг | bug | Грешка в поведението или логиката. |
| компилатор | compiler | Инструмент, който проверява и превежда кода. |
