# Урок 2: Първа C# програма

## Какво ще научиш

До края на този урок ще можеш да:

- разбереш какво е конзолна програма;
- напишеш първата си C# инструкция;
- покажеш текст на екрана;
- прочетеш прост C# код отгоре надолу;
- откриваш малки синтактични грешки;
- променяш готова програма без страх.

## Защо това е важно

В предишния урок мислехме като програмисти: стъпки, ред, точни инструкции.

Сега ще направим следващата крачка: ще дадем истинска инструкция на компютъра с C#.

Първата ни цел е малка:

```text
Кажи на програмата да покаже текст.
```

Това може да изглежда просто, но е важна основа. Ако можеш да покажеш текст, можеш да видиш какво
прави програмата. А когато виждаш какво прави програмата, можеш да мислиш, тестваш и поправяш.

## История с Byte

Byte вече знае, че трябва да следва точни стъпки.

Днес му даваме първата истинска команда на C#:

```csharp
Console.WriteLine("Здравей, Byte!");
```

Byte я разглежда внимателно и казва:

> Byte: "Виждам точна инструкция. Има име на действие, скоби, текст в кавички и знак за край. Това мога да изпълня."

Точно така. C# има правила за писане. Тези правила се наричат синтаксис.

Синтаксисът е като правопис за компютъра. Ако липсва важен символ, компютърът може да не разбере
какво искаме.

## Основно обяснение

В този курс започваме с конзолни програми.

Конзолата е текстов прозорец. В него програмата може да показва текст и по-късно ще може да чете
какво пише потребителят.

Можеш да си я представиш така:

```text
+----------------------------------+
| Конзола                          |
|                                  |
| Здравей, свят!                   |
|                                  |
+----------------------------------+
```

Първата C# инструкция, която ще използваме, е:

```csharp
Console.WriteLine("Здравей, свят!");
```

Тя означава:

```text
Покажи ред текст в конзолата.
```

Нека погледнем частите:

- `Console` е мястото, свързано с конзолата.
- `WriteLine` означава "напиши ред".
- Текстът между кавичките е това, което ще се покаже.
- `;` казва "тази инструкция свършва тук".

Не е нужно още да знаеш всичко за `Console` и `WriteLine`. Засега ги приеми като готова команда
за показване на текст.

## Визуално обяснение

Ето как мислим за първата програма:

```text
[ C# инструкция ] -> [ компютърът я изпълнява ] -> [ текст в конзолата ]
```

С конкретен пример:

```text
Console.WriteLine("Здравей!");
          |
          v
+----------------+
| Здравей!       |
+----------------+
```

Ако имаме няколко инструкции, те се изпълняват отгоре надолу:

```text
Код:

1. Покажи "Старт"
2. Покажи "Byte е готов"
3. Покажи "Край"

Изход:

Старт
Byte е готов
Край
```

Редът на инструкциите е редът на резултата.

## Първи пример

Ето първата ни малка C# програма:

```csharp
Console.WriteLine("Здравей, свят!");
```

Когато програмата се изпълни, в конзолата ще видиш:

```text
Здравей, свят!
```

Можем да покажем и повече от един ред:

```csharp
Console.WriteLine("Здравей!");
Console.WriteLine("Аз уча C#.");
Console.WriteLine("Byte също е тук.");
```

Резултат:

```text
Здравей!
Аз уча C#.
Byte също е тук.
```

Забележи нещо важно: всяко `Console.WriteLine(...)` показва нов ред.

## Разбивка на кода

Нека разгледаме тази инструкция:

```csharp
Console.WriteLine("Здравей, Byte!");
```

Разбивка:

- `Console` казва, че работим с конзолата.
- `.` свързва `Console` с действието `WriteLine`.
- `WriteLine` е действието: напиши ред.
- `(` и `)` ограждат това, което подаваме на действието.
- `"Здравей, Byte!"` е текстът, който ще се покаже.
- `;` завършва инструкцията.

Можем да го видим така:

```text
+---------+    +-----------+    +-------------------+
| Console | -> | WriteLine | -> | "Здравей, Byte!"  |
+---------+    +-----------+    +-------------------+
                                      |
                                      v
                              показва се в конзолата
```

Най-важните символи за този урок са:

```text
" "   кавички около текста
( )   скоби около това, което WriteLine получава
;     край на инструкцията
```

Byte ги харесва, защото му казват точно къде започва и къде свършва всяка част.

## Чести грешки

### Грешка 1: Липсваща точка и запетая

```csharp
Console.WriteLine("Здравей")
```

Защо е проблем:

C# очаква `;` в края на инструкцията.

Поправка:

```csharp
Console.WriteLine("Здравей");
```

### Грешка 2: Липсваща кавичка

```csharp
Console.WriteLine("Здравей);
```

Защо е проблем:

Текстът започва с кавичка, но не завършва с кавичка. Компютърът не знае къде свършва текстът.

Поправка:

```csharp
Console.WriteLine("Здравей");
```

### Грешка 3: Текст без кавички

```csharp
Console.WriteLine(Здравей);
```

Защо е проблем:

Когато искаме да покажем текст, го слагаме в кавички.

Поправка:

```csharp
Console.WriteLine("Здравей");
```

### Грешка 4: Объркани малки и главни букви

```csharp
console.writeline("Здравей");
```

Защо е проблем:

C# прави разлика между главни и малки букви. `Console.WriteLine` трябва да бъде написано точно.

Поправка:

```csharp
Console.WriteLine("Здравей");
```

### Грешка 5: Очакваме програмата да промени текста сама

```csharp
Console.WriteLine("Кажи нещо интересно");
```

Защо е проблем:

Компютърът няма да измисли интересно съобщение. Той ще покаже точно текста между кавичките.

Резултат:

```text
Кажи нещо интересно
```

Byte би казал:

> Byte: "Текстът беше инструкция за мен само ако е извън кавичките. В кавичките е нещо за показване."

## Как мисли програмистът

Когато видиш C# код, не бързай да го стартираш.

Първо го прочети като инструкции:

```csharp
Console.WriteLine("Едно");
Console.WriteLine("Две");
Console.WriteLine("Три");
```

Мисленето може да бъде:

```text
1. Първата инструкция показва "Едно".
2. Втората инструкция показва "Две".
3. Третата инструкция показва "Три".
4. Значи изходът ще бъде три реда.
```

После можеш да сравниш предположението си с реалния резултат.

Това е много ценен навик:

```text
прочети -> предвиди -> стартирай -> сравни
```

Ако резултатът е различен, това не е провал. Това е следа.

## Практика

### Какво ще отпечата тази програма?

```csharp
Console.WriteLine("Старт");
Console.WriteLine("Byte проверява инструкциите.");
Console.WriteLine("Готово");
```

Твоето предположение:

```text
...
```

Провери реда. Програмата чете отгоре надолу.

### Намери бъга

```csharp
Console.WriteLine("Здравей, Byte!")
Console.WriteLine("Добре дошъл.");
```

Какъв е проблемът?

Подсказка:

Първата инструкция не е завършена както C# очаква.

### Поправи кода

Програмата трябва да покаже:

```text
Аз уча C#.
Това е първата ми програма.
```

Но кодът има грешки:

```csharp
Console.WriteLine(Аз уча C#.);
Console.WriteLine("Това е първата ми програма.)
```

Поправи кода така, че текстът да се покаже правилно.

## Мини проект

### Конзолна визитка

Направи първата си малка завършена програма: конзолна визитка.

Програмата трябва да покаже:

- име;
- възраст;

```text

3. Колко Console.WriteLine инструкции ми трябват?
4. Има ли всеки текст кавички?
5. Има ли всяка инструкция точка и запетая?
```

## Обобщение

В този урок научи:

- конзолната програма показва текст в текстов прозорец;
- `Console.WriteLine` показва един ред текст;
- C# има точен синтаксис;
- текстът за показване се пише в кавички;
- инструкцията завършва с `;`;
- програмата изпълнява инструкциите отгоре надолу;
- преди стартиране е полезно да предвидиш резултата.

Най-важната идея:

```text
Кодът е точна инструкция, а текстът в кавички е това, което ще се покаже.
```
