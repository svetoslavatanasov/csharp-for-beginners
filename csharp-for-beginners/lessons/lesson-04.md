# Урок 4: Вход от клавиатурата и преобразуване на типове

## Какво ще научиш

До края на този урок ще можеш да:

- прочетеш текст от клавиатурата с `Console.ReadLine`;
- запазиш въведеното в променлива;
- разбереш защо входът от клавиатурата първо идва като текст;
- преобразуваш текст към цяло число с `int.Parse`;
- преобразуваш текст към дробно число с `double.Parse`;
- пишеш малки интерактивни конзолни програми.

## Защо това е важно

Досега програмите ни използваха стойности, които ние написахме предварително в кода.

Но много програми трябва да попитат човека:

- Как се казваш?
- На колко години си?
- Колко точки имаш?
- Каква цена искаш да въведеш?

Входът от клавиатурата прави програмата интерактивна. Тя вече не само говори. Тя слуша.

## История с Byte

Byte иска да поздрави човека по име.

Първо пробва така:

```csharp
Console.WriteLine("Здравей, човек!");
```

Това работи, но не е лично.

Byte пита:

> Byte: "Мога ли първо да попитам за име, после да използвам отговора?"

Да. За това ни трябва `Console.ReadLine()`.

Но Byte трябва да помни още нещо важно:

> Byte: "Всичко, което човекът напише от клавиатурата, пристига при мен като текст."

Дори ако човекът напише `15`, програмата първо го получава като текст `"15"`.

## Основно обяснение

За да прочетем ред от клавиатурата, използваме:

```csharp
Console.ReadLine()
```

Най-често го пазим в променлива:

```csharp
string name = Console.ReadLine();
```

Този ред означава:

```text
Изчакай човекът да напише нещо.
Когато натисне Enter, вземи написания текст.
Сложи го в променливата name.
```

Обикновено преди това показваме въпрос:

```csharp
Console.WriteLine("Как се казваш?");
string name = Console.ReadLine();

Console.WriteLine("Здравей!");
Console.WriteLine(name);
```

Ако човекът напише:

```text
Мая
```

резултатът ще бъде:

```text
Как се казваш?
Здравей!
Мая
```

### Входът е текст

`Console.ReadLine()` връща текст.

Това значи, че този код е правилен:

```csharp
string ageText = Console.ReadLine();
```

Но този код не е правилен:

```csharp
int age = Console.ReadLine();
```

Причината:

```text
Console.ReadLine() дава string.
int очаква цяло число.
```

Ако искаме число, трябва да преобразуваме текста.

## Визуално обяснение

Входът от клавиатурата минава през програмата така:

```text
[ клавиатура ]
      |
      v
Console.ReadLine()
      |
      v
[ string променлива ]
```

Пример:

```text
човекът пише: 15
      |
      v
Console.ReadLine()
      |
      v
ageText = "15"
```

Ако ни трябва число, добавяме преобразуване:

```text
ageText = "15" -> int.Parse(ageText) -> age = 15
```

Byte го мисли така:

```text
"15"   е текст
15     е число
```

Изглеждат почти еднакво за човек, но за C# са различни видове стойности.

## Първи пример

Нека направим програма, която пита за име:

```csharp
Console.WriteLine("Как се казваш?");
string name = Console.ReadLine();

Console.WriteLine("Здравей!");
Console.WriteLine(name);
```

Примерно изпълнение:

```text
Как се казваш?
Иво
Здравей!
Иво
```

Това вече е интерактивна програма. Резултатът зависи от това какво въведе човекът.

## Разбивка на кода

```csharp
Console.WriteLine("Как се казваш?");
```

Това показва въпрос.

```csharp
string name = Console.ReadLine();
```

Това прочита отговора и го пази като текст.

```csharp
Console.WriteLine(name);
```

Това показва стойността, която човекът е въвел.

### Преобразуване към число

Ако искаме да прочетем възраст, първо получаваме текст:

```csharp
Console.WriteLine("На колко години си?");
string ageText = Console.ReadLine();
```

После преобразуваме текста към цяло число:

```csharp
int age = int.Parse(ageText);
```

Цял пример:

```csharp
Console.WriteLine("На колко години си?");
string ageText = Console.ReadLine();
int age = int.Parse(ageText);

int nextYear = age + 1;

Console.WriteLine("След една година ще си на:");
Console.WriteLine(nextYear);
```

Ако човекът въведе:

```text
15
```

резултатът ще бъде:

```text
След една година ще си на:
16
```

В този пример използваме малко събиране с `+`. В следващия урок ще разгледаме операторите по-спокойно.

### Преобразуване към дробно число

За дробно число използваме `double.Parse`:

```csharp
Console.WriteLine("Въведи височина в метри:");
string heightText = Console.ReadLine();
double height = double.Parse(heightText);

Console.WriteLine("Въведената височина е:");
Console.WriteLine(height);
```

В примерите от курса ще пишем дробни числа с точка, например `1.75`. Ако в твоята среда това не се приеме, пробвай със запетая: `1,75`. Засега не е нужно да помниш защо има разлика. Важното е идеята: текстът трябва да стане дробно число.

## Чести грешки

### Грешка 1: Опит да сложим текст директно в `int`

```csharp
int age = Console.ReadLine();
```

Защо е проблем:

`Console.ReadLine()` връща текст, а `int` очаква цяло число.

Поправка:

```csharp
string ageText = Console.ReadLine();
int age = int.Parse(ageText);
```

### Грешка 2: Преобразуване на дума към число

```csharp
string ageText = "петнадесет";
int age = int.Parse(ageText);
```

Защо е проблем:

`int.Parse` очаква текст, който изглежда като цяло число, например `"15"`.

Поправка:

```csharp
string ageText = "15";
int age = int.Parse(ageText);
```

### Грешка 3: Кавички около името на променливата

```csharp
string ageText = "15";
int age = int.Parse("ageText");
```

Защо е проблем:

Това се опитва да преобразува текста `"ageText"`, не стойността в променливата.

Поправка:

```csharp
string ageText = "15";
int age = int.Parse(ageText);
```

### Грешка 4: Забравени скоби

```csharp
string name = Console.ReadLine;
```

Защо е проблем:

`ReadLine` е действие. За да го извикаме, пишем скоби.

Поправка:

```csharp
string name = Console.ReadLine();
```

### Грешка 5: Очакваме програмата да поправи входа сама

Ако програмата пита за число, а човекът напише:

```text
много
```

`int.Parse` няма да знае как да направи число от това.

Засега ще приемаме, че човекът въвежда правилен тип стойност. По-късно, когато учим условия, ще
проверяваме входа по-внимателно.

## Как мисли програмистът

Когато пишеш програма с вход, мисли в този ред:

```text
1. Какво трябва да попитам?
2. Какво ще въведе човекът?
3. Текст ли ми трябва, или число?
4. Ако ми трябва число, кога ще преобразувам текста?
5. Как ще използвам стойността?
```

Пример:

```text
Задача: Попитай за възраст и покажи възрастта след една година.

Вход: възраст, написана от клавиатурата
Първо идва като: string
Трябва ми като: int
Стъпки:
1. Покажи въпрос.
2. Прочети текст.
3. Преобразувай към int.
4. Добави 1.
5. Покажи резултата.
```

Byte би казал:

> Byte: "Първо слушам. После решавам дали текстът трябва да стане число."

## Мини проект

### Интерактивна визитка

Направи програма, която пита човека за:

- име;
- възраст;
- град;
- хоби;
- любимо число.

Преобразувай възрастта и любимото число към `int`.

## Тест за самопроверка

1. Какво прави `Console.ReadLine()`?
2. Какъв тип стойност връща `Console.ReadLine()`?
3. Защо `int age = Console.ReadLine();` е проблем?
4. За какво служи `int.Parse`?
5. За какво служи `double.Parse`?
6. Каква е разликата между `"15"` и `15`?
7. Какво ще стане, ако `int.Parse` получи текст като `"Byte"`?
8. Защо засега приемаме, че потребителят въвежда правилни стойности?

## Обобщение

В този урок научи:

- `Console.ReadLine()` чете един ред от клавиатурата;
- входът първо идва като `string`;
- ако ни трябва цяло число, използваме `int.Parse`;
- ако ни трябва дробно число, използваме `double.Parse`;
- кавичките променят смисъла: `"15"` е текст, а `15` е число;
- добрата интерактивна програма първо пита ясно, после чете отговора.

Най-важната идея:

```text
Входът от клавиатурата започва като текст.
Преобразуваме го само когато ни трябва число.
```
